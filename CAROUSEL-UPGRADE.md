## Music Strategy

**Must add background music to every carousel.** This triggers the Reels algorithm.

Recommended: Instrumental/lo-fi beats (won't distract from reading). Browse Instagram's music library for:
- Trending lo-fi
- Trending instrumental
- Any currently viral sound (piggyback on trend)

**Length:** Match to ~15-30 seconds expected swipe time.

## Quick Template Structure

Each carousel slide uses the same visual frame (dark bg, @rpedroni branding) but:
- **Slide 1:** Larger text, bolder color accent, no @handle (let the hook breathe)
- **Middle slides:** Normal layout, supporting text/data/visuals
- **Last slide:** CTA with @rpedroni, bright accent color, action verb

This maintains brand consistency while optimizing for the carousel format.
